# Investor Demo Runbook

## Quick Start — Demo Mode (Zero API Calls)

Navigate to:

```
https://www.archiaisolution.pro/?demo=true
```

Or locally:

```
http://localhost:3000/?demo=true
```

This loads a pre-generated A1 architectural sheet (real AI output, not a mockup) and skips directly to the results view.

### What the investor sees

1. **A1 Sheet Viewer** — Complete architectural sheet with floor plans, elevations, sections, and 3D views
2. **Panel Gallery** — 8 individual SVG technical drawings (click to expand)
3. **Demo Banner** — Clear label: "Viewing pre-generated demo — real AI output from a 150 m² London home"
4. **Export Options** — Download A1 sheet as PNG, CAD/BIM export panel
5. **AI Modify Panel** — Shows "available in live generation mode" placeholder

### Key talking points

- "This sheet was generated in under 3 minutes from a text brief"
- "8 technical drawings are generated as deterministic SVGs — dimensionally accurate"
- "4 photorealistic panels are generated with FLUX AI"
- "The Design DNA system ensures 98%+ consistency across all views"
- "Total cost per sheet: approximately $0.10"

## Live Generation Mode

Click **"Start Designing"** on the landing page (or **"Try Live Generation"** from the demo view).

### Prerequisites

- `npm run dev` running locally (React on :3000, Express on :3001)
- Valid `TOGETHER_API_KEY` in `.env` (requires paid tier)
- Valid `REACT_APP_GOOGLE_MAPS_API_KEY` and `REACT_APP_OPENWEATHER_API_KEY`

### Walk-through (5 steps)

1. **Location** — Enter an address or use geolocation. Draw site boundary (optional).
2. **Intelligence** — Review climate data, zoning, and style recommendations. Click Next.
3. **Portfolio** — Upload reference images (optional). Adjust style blend weights. Click Next.
4. **Specifications** — Select building type (e.g., Residential > Detached House). Enter area (e.g., 150 m²). Click "Generate Program" to auto-create room schedule. Click Next.
5. **Generate** — Generation starts automatically. Progress shows 5 stages.

### Expected timing

- DNA generation: ~15 seconds
- Panel rendering: ~2-3 minutes (8 SVGs + 4 FLUX API calls with rate limiting)
- Composition: ~10 seconds
- **Total: ~3 minutes**

### What to say during the wait

- "The system is generating Design DNA — extracting exact dimensions, materials, and room layouts"
- "Technical drawings are rendered as vector SVGs for dimensional accuracy"
- "3D views are generated by FLUX AI with the DNA as a consistency lock"

## Troubleshooting

| Issue                    | Cause                            | Fix                                       |
| ------------------------ | -------------------------------- | ----------------------------------------- |
| "API rate limit reached" | Together.ai 429 error            | Wait 60 seconds, retry                    |
| "API key invalid"        | Missing/expired TOGETHER_API_KEY | Check `.env`                              |
| Generation stuck >5 min  | API congestion                   | Refresh and retry, or switch to demo mode |
| No map/location          | Missing Google Maps key          | Check `REACT_APP_GOOGLE_MAPS_API_KEY`     |
| Blank A1 sheet           | Composition endpoint unreachable | Ensure `npm run server` is running        |

## Fallback Plan

If live generation fails during the meeting:

1. Navigate to `/?demo=true` to show the pre-generated output
2. Explain: "The live generation occasionally hits rate limits — here's a real output from an earlier run"
3. Walk through the A1 sheet, panel gallery, and export options
